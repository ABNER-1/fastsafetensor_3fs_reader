# fastsafetensor_3fs_reader 测试用例梳理

> 更新时间: 2026/3/11  
> 总计: 2 个测试文件，29（CI）+ 25（3FS）= 54 个测试用例

## 目录结构

```
tests/
├── conftest.py          # 共享 fixtures
├── test_mock.py         # [CI] MockFileReader 完整行为测试，无需外部依赖
└── test_threefs.py      # [3FS] ThreeFSFileReader 真实接口测试，需要 3FS 环境
```

---

## conftest.py — 共享 Fixtures

| Fixture | 作用 | 使用方 |
|---------|------|--------|
| `tmp_safetensors` | 单个最小 safetensors 文件（2×3 F32 zeros） | mock / 3FS |
| `tmp_safetensors_multi` | 3 个 safetensors 文件，各含不同 tensor 名 | mock / 3FS |
| `tmp_safetensors_large` | 含已知非零数据的文件（256 个 F32，值为 range(256)），返回 `(path, raw_bytes)` | mock / 3FS |
| `tmp_model_shards` | 5 个分片文件，模拟真实模型加载场景，返回 `[(path, raw_bytes)]` | mock / 3FS |
| `mock_reader` | `MockFileReader` 实例，测试结束后自动 close | mock |
| `threefs_mount_point` | 从 `THREEFS_MOUNT_POINT` 环境变量读取挂载点，未设置则 skip | 3FS |
| `threefs_reader` | `ThreeFSFileReader` 实例，C++ 模块不可用或未设置挂载点则 skip | 3FS |

**运行条件**：无需任何外部依赖，所有 CI 环境均可运行。

